# ImageScope - 快捷键参考

## 导航 & 缩放

| 快捷键 | 功能 |
|--------|------|
| **滚轮** | 在鼠标位置放大/缩小 |
| **Ctrl + 滚轮**<br/>**Cmd + 滚轮** | 同上（macOS 兼容） |
| **Ctrl + Plus** | 从中心放大 |
| **Ctrl + Minus** | 从中心缩小 |
| **Ctrl + 0** | 缩放到适应窗口 |
| **双击** | 切换 100% 缩放和适应窗口 |

## 平移 & 拖动

| 快捷键 | 功能 |
|--------|------|
| **左键拖动** | 平移图像 |
| **中键拖动** | 平移图像 |
| **Space + 左键拖动** | 平移图像（即使在 ROI 模式下也可用） |
| **Ctrl + 左键拖动** | 框选后放大到选中区域 |

## 像素操作

| 快捷键 | 功能 |
|--------|------|
| **Alt + 左键点击** | 复制像素颜色（格式：`#RRGGBB RGB(r,g,b)`） |
| **鼠标悬停** | 显示实时像素坐标和颜色值 |

## ROI & 分析

| 快捷键 | 功能 |
|--------|------|
| **R / r** | 开启/关闭 ROI 选择模式 |
| **（ROI 模式中）左键拖动** | 绘制 ROI 矩形 |
| **G / g** | 开启/关闭像素网格（≥ 800% 缩放时显示） |
| **H / h** | 开启/关闭直方图分析面板 |
| **L / l** | 开启/关闭测量工具 |

## 通道查看

| 快捷键 | 功能 |
|--------|------|
| **1** | RGB 彩色通道 |
| **2** | 仅显示红色通道 (R) |
| **3** | 仅显示绿色通道 (G) |
| **4** | 仅显示蓝色通道 (B) |
| **5** | 仅显示 Alpha 通道 (A) |

## 显示模式

| 快捷键 | 功能 |
|--------|------|
| **D / d** | 切换灰度模式 |
| **X / x** | 开启/关闭十字线光标 |
| **B / b** | 循环切换背景：棋盘格 → 白色 → 黑色 → 灰色 |
| **M / m** | 开启/关闭小地图（显示整体位置） |

## 图像变换

| 快捷键 | 功能 |
|--------|------|
| **F / f** | 水平翻转 |
| **Shift + F** | 垂直翻转 |
| **N / n** | 顺时针旋转 90° |

## 使用场景示例

### 精确像素检查工作流
1. 打开图像
2. `Ctrl + 左键拖动` → 框选要检查的区域
3. `鼠标悬停` → 查看像素坐标和 RGB 值
4. `Alt + 左键点击` → 复制像素颜色
5. `G` → 启用像素网格看清每个像素

### 分析工作流
1. 打开图像
2. `H` → 显示直方图
3. `1-5` → 切换不同通道查看
4. `R` → 开启 ROI 模式
5. `左键拖动` → 框选感兴趣区域
6. 查看侧边栏的 ROI 统计信息

### 演示/检查工作流
1. `Ctrl + 0` → 缩放到适应窗口
2. `B` → 调整背景对比
3. `X` → 启用十字线
4. `M` → 显示小地图方便导航
5. `滚轮` 或 `Ctrl + ±` → 精确缩放

### 图像检查工作流
1. `F` → 水平翻转（对称性检查）
2. `Shift + F` → 垂直翻转
3. `N` → 旋转 90°（检查不同方向）
4. 结合 `L` → 使用测量工具量取尺寸

## 提示

- 所有模式（像素网格、ROI、直方图等）可以同时使用
- 快捷键不区分大小写（`G` 和 `g` 功能相同）
- 在 ROI 模式下，使用 `Space + 左键拖动` 可以平移而不中断 ROI 选择
- 框选放大（`Ctrl + 左键拖动`）在非 ROI 模式下可用
